# rustium

Rustium's standalone CDC service and command-line interface. It loads native YAML or Debezium-style properties, runs native PostgreSQL, MySQL, SQL Server, Oracle, and MongoDB CDC, and integrates every remaining Debezium database source through the durable HTTP/Kafka bridge.

See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) for configuration, deployment, and embedding instructions. The normative architecture is in [docs/design.md](https://github.com/ulnit/rustium/blob/main/docs/design.md).

## 简体中文

Rustium 的独立 CDC 服务和命令行程序。它加载原生 YAML 或 Debezium 风格 properties，运行原生 PostgreSQL、MySQL、SQL Server、Oracle、MongoDB CDC，并通过持久 HTTP/Kafka bridge 集成其余全部 Debezium 数据库 Source。
