# rustium

Rustium's standalone CDC service and command-line interface. It loads native YAML or Debezium-style properties, runs PostgreSQL, MySQL, or SQL Server CDC, and delivers typed events to stdout or Kafka.

See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) for configuration, deployment, and embedding instructions. The normative architecture is in [docs/design.md](https://github.com/ulnit/rustium/blob/main/docs/design.md).

## 简体中文

Rustium 的独立 CDC 服务和命令行程序。它加载原生 YAML 或 Debezium 风格 properties，运行 PostgreSQL、MySQL 或 SQL Server CDC，并将类型化事件投递到 stdout 或 Kafka。

配置、部署和嵌入步骤请参阅[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)，规范架构请参阅[设计文档](https://github.com/ulnit/rustium/blob/main/docs/design.md)。
