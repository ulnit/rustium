# rustium-column-transform

Shared Debezium-compatible column transformation engine used by Rustium source connectors.

Rustium 源连接器共享的、兼容 Debezium 的列转换引擎。

This crate is an internal implementation component. Applications should configure transformations through the source connector configuration rather than depending on it directly.

此 crate 是内部实现组件。应用程序应通过源连接器配置列转换，不应直接依赖它。
