# rustium-core

Rustium's typed event model, source and sink traits, replayable source positions, bounded runtime, signal acknowledgement, and checkpoint ordering contracts.

Use this crate to implement an embedded source, sink, encoder, or checkpoint store while preserving Rustium's at-least-once guarantees. See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) and [design](https://github.com/ulnit/rustium/blob/main/docs/design.md).

## 简体中文

Rustium 的类型化事件模型、Source/Sink trait、可重放源位点、有界 runtime、信号确认以及 checkpoint 顺序契约。

实现嵌入式 source、sink、encoder 或 checkpoint store 且保持 Rustium 至少一次投递保证时使用此 crate。详见[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)和[设计文档](https://github.com/ulnit/rustium/blob/main/docs/design.md)。
