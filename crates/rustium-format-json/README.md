# rustium-format-json

Rustium native JSON, Debezium JSON, and Confluent JSON Schema encoders. It preserves typed source metadata, transaction fields, schema descriptors, and optional delete tombstones.

Use this crate for JSON delivery or with the Kafka Schema Registry sink. See the [project README](https://github.com/ulnit/rustium/blob/main/README.md).

Checked-in PostgreSQL, MySQL, and SQL Server fixtures protect the complete Debezium JSON and JSON Schema contracts.

Rustium 原生 JSON、Debezium JSON 和 Confluent JSON Schema 编码器，保留类型化 source 元数据、事务字段、schema 描述和可选 delete tombstone。

用于 JSON 投递或配合 Kafka Schema Registry sink。详见[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)。

仓库内置 PostgreSQL、MySQL 和 SQL Server fixture，用于保护完整的 Debezium JSON 与 JSON Schema 契约。
