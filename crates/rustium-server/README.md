# rustium-server

Rustium's HTTP management, health, lifecycle, signal, and Prometheus metrics server. It exposes bounded mutation routes and low-cardinality operational status for an embedded connector runtime.

See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) for endpoint and deployment contracts.

## 简体中文

Rustium 的 HTTP 管理、健康检查、生命周期、信号和 Prometheus metrics server，为嵌入式 connector runtime 提供有界变更端点和低基数运行状态。

端点和部署契约请参阅[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)。
