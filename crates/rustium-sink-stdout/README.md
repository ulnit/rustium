# rustium-sink-stdout

Development stdout sink for Rustium. It is useful for local inspection and examples, but it is best-effort and must not be used as a production durability boundary.

See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) for embedding examples.

## 简体中文

Rustium 的开发用 stdout sink，适合本地检查和示例，但属于 best-effort，不能作为生产持久性边界。

嵌入示例请参阅[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)。
