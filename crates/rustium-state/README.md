# rustium-state

Versioned SQLite checkpoint storage for Rustium. It persists source positions, connector state, configuration fingerprints, and generations atomically while retaining upgrade and replay compatibility.

See the [project README](https://github.com/ulnit/rustium/blob/main/README.md) and [upgrade guide](https://github.com/ulnit/rustium/blob/main/docs/upgrades.md).

## 简体中文

Rustium 的版本化 SQLite checkpoint storage，原子持久化 source position、connector state、配置指纹和 generation，并保留升级与重放兼容性。

详见[项目 README](https://github.com/ulnit/rustium/blob/main/README.md)和[升级指南](https://github.com/ulnit/rustium/blob/main/docs/upgrades.md)。
